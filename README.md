## FAQ

#### What is this for?

This is a proof of concept of the reactive mutability of DOM elements accomplished with the use of `@mutablejs/core` and `@mutablejs/dom`

## Installation and Startup

With **npm**:

```bash
    npm install

	npm run start
```

With **yarn**:

```bash
    yarn

	yarn start
```
